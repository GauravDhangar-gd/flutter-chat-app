# Flutter Chat App

A modern WhatsApp-inspired chat application built using Flutter and Firebase.

## Features

- Firebase Authentication
- User Profiles
- Real-time Messaging
- Online / Offline Status
- Last Seen
- Typing Indicator
- Read Receipts
- Image Messages
- Voice Messages
- Emoji Picker
- Reply Messages
- Message Reactions
- Delete for Me
- Delete for Everyone
- Message Search
- Chat Wallpaper
- Profile Photo Upload
- Cloudinary Integration

## Tech Stack

- Flutter
- Dart
- Firebase Auth
- Cloud Firestore
- Firebase Storage
- Cloudinary
- Shared Preferences
